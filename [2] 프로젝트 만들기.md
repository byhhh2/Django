### 프로젝트 생성

```

(in venv) $ django-admin startproject [project-name]

```

<br />

### 프로젝트 구조

```

project-name [1]/
    manage.py
    project-name/
        __init__.py
        settings.py
        urls.py
        asgi.py
        wsgi.py

```

<br />

### 프로젝트 실행

```

// in project folder [1]

// localhost:8000
(in venv) $ python manage.py runserver

// shell
(in venv) $ python manage.py shell

```
