## Django

<br />

### 가상환경 만들기

```

[가상환경으로 지정할 폴더 안]

$ pipenv --three // python3이 설치된 가상환경으로
$ pipenv shell // 활성화, 가상환경이 없으면 만들고 활성화
$ exit // 가상환경 종료

```

<br />

### 장고를 가상환경에 설치

```

(in venv) $ pipenv install django

```

<br />
